Este repositorio contiene la solución algorítmica desarrollada en Python para el **Problema 2** de la guía de actividades.

##  Descripción del Problema
El sistema gestiona de manera estructurada los precios de un menú de restaurante representado a través de una estructura matricial (`Listas de listas`). Aplica de manera automática un descuento del **15%** a los productos que cumplan simultáneamente con una lógica de negocio específica.

### Lógica de Negocio Adoptada:
* **Categoría Objetivo:** Platos Fuertes
* **Umbral de Precio Mínimo:** $20,000 COP
* **Descuento Aplicado:** 15% si (Categoría == "Platos Fuertes" Y Precio > 20,000)

* **Enfoque:** Programación Estructurada y Modular
* **Entorno de Desarrollo:** Visual Studio Code
